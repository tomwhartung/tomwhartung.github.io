Adaptive Resume

I have used device detection and jQuery Mobile to create an adaptive, interactive, and comprehensive online resume, and keep the source for it on github.

When viewed on a desktop or tablet, the resume appears as a single page.

When viewed on a phone, the server:

  • Adds the jQuery Mobile JavaScript library to the code served to the browser
  • Breaks each section into a separate screen
  • Adds navigation bars at the top and bottom of each screen, allowing the user to focus on one section of the resume at a time
idMyGadget
The resume uses an Adapter API I wrote named idMyGadget to detect the visitor's device and serve appropriate content.
my resume
The source for my resume is in the online resume repo. These repos include documentation that should be enough to enable you to create your own adaptive, interactive, and comprehensive online resume.

Regardless of the device, the resume uses JavaScript to make the experience more interactive. And although it lists all of my jobs — and is hence comprehensive — it will show all of them only if the visitor really wants to drill back into my past.

When going on job interviews, it sometimes seems that they want you to have already solved the problem(s) they need to solve themselves. This device detection, AWD, and RESS work is my attempt to meet those needs3.
