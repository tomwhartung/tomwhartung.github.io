Adaptive Resume

I have used device detection and jQuery Mobile to create an adaptive, interactive, and comprehensive online resume, and keep the source to it on github.

When viewed on a desktop or tablet, the resume appears as a single page. When viewed on a phone, the server:

  • adds the jQuery Mobile library to the code served to the browser
  • breaks the single page into a series of mobile-phone-sized screens
  • adds navigation bars at the top and bottom of each screen, allowing the user to focus on one section of the resume at a time

Regardless of the device, the resume uses JavaScript to make the experience more interactive. Although it lists all of my jobs — and thus is comprehensive — it does so only if the visitor really wants to drill back into my past.
